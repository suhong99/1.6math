# 작업 로그 템플릿 및 작성 규칙

## 파일 명명 규칙

```
YYYY-MM-DD-작업명.md
```

예: `2026-04-03-피드백기능추가.md`

## 템플릿

```markdown
# YYYY-MM-DD 작업명

## 상태

진행중 | 완료 | 중단

## 목표

- 이 작업에서 달성하려는 것

## 관련 이슈 / PR

- Issue #번호
- PR #번호 (완료 시)

## 관련 파일

- `src/feature/xxx/...`
- `src/app/...`

## 작업 단계

- [ ] 단계 1
- [ ] 단계 2
- [ ] 단계 3

## 진행 상황

### YYYY-MM-DD

- 작업 내용 기록

## 중단 시 이어서 할 작업

(작업 중단 시 다음 단계를 여기에 기록)

## 완료 요약

(작업 완료 시 주요 변경 사항 요약)
```
