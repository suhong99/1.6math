오늘 날짜와 작업명을 바탕으로 `docs/work-logs/YYYY-MM-DD-작업명.md` 파일을 생성하세요.

## 단계

1. 오늘 날짜 확인 (시스템 컨텍스트의 currentDate 사용)
2. 작업명은 사용자 요청에서 간결하게 추출 (한글 가능, 예: `패치파서수정`, `캐릭터이미지업로드`)
3. `docs/work-logs/README.md` 템플릿을 참고하여 파일 생성
4. 파일 내용: 목표, 관련 브랜치, 예상 실행 스크립트 포함

## 파일 구조

```markdown
# [작업명]

## 개요

- **시작일**: YYYY-MM-DD
- **상태**: 진행중
- **관련 브랜치**: feature/XX

## 목표

- [ ] 목표 1

## 진행 상황

### YYYY-MM-DD

- 작업 시작

## 실행 스크립트

(해당 작업에 필요한 스크립트)

## 중단 시 이어서 할 작업

- [ ] 다음 단계

## 관련 파일

- 관련 파일 목록
```

파일 생성 후 작업을 진행하세요.
